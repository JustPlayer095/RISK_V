В архиве - основные модули для работы с OSDP (процессор LPC-1227).

Дополнительно в основной программе необходимо раз в 10 мсек опрашивать
логические входы (в данном случае - датчики автоматики ворот, 4 штуки).
В коде PR-U07 это уже организовано - в дополнение надо только класть
новые значения в структуру статуса входов и взводить флаг изменения,
по которому при полинге драйвер OSDP отправит их хосту).

Также драйверу OSDP необходимо при полинге отправлять код тага, передаваемый 
на контроллер (опять же по дополнителному собственному флагу, который
взведет считыватель, а сбросит драйвер после отправки).

Кроме прочего, OSDP должен уметь отправлять по запросу ID report, Capability 
report, а еще общую конфигурацию, которую надо сформировать.

Для обычного ридера конфигурация в файле AppTypes.h.
