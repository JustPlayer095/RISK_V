# UART Simple Project для K1921VG015

## Описание
Простейший проект для вывода информации в COM-порт через UART на микроконтроллере K1921VG015.

## Функциональность
- Вывод системной информации при запуске
- Периодический вывод счетчика и времени работы
- Мигание светодиодами для индикации работы
- UART на скорости 115200 бод

## Настройки UART
- **Скорость**: 115200 бод
- **Формат**: 8N1 (8 бит данных, без четности, 1 стоп-бит)
- **GPIO**: 
  - TX - A.3
  - RX - A.2

## Сборка проекта
```bash
make all
```

## Очистка
```bash
make clean
```

## Отладка
```bash
make debug
```

## Вывод программы
При запуске программа выводит:
1. Информацию о системе (частота, UID)
2. Каждые 2 секунды:
   - Счетчик итераций
   - Время работы в секундах
   - Статус работы

## Требования
- Компилятор RISC-V GCC
- Отладочный адаптер (FTDI)
- Терминал для COM-порта (115200 бод)
